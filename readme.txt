# Sistema de Gestión de Recursos Humanos 

## Descripción
Sistema de análisis de RRHH desarrollado con R Shiny

## Características
- Análisis de personal
- Evaluación de desempeño
- Predicción de rotación
- Dashboard ejecutivo

## Cómo usar
1. Instalar R y RStudio
2. Instalar paquetes necesarios
3. Ejecutar PROYECTO - RRHH.R
4. Cargar datos_ejemplo_rrhh.csv
